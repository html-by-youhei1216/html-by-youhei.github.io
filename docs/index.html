主な機能

📚 HTML学習コース

12章の内蔵レッスン。解説・練習・クイズ付き。進捗はブラウザに自動保存。

💻 HTMLエディタ

シンタックスハイライト、行番号、タグ補完、コード整形、ミス検出。

👁 リアルタイムプレビュー

入力と同時に更新。パソコン・タブレット・スマホ表示の切替。

✏️ 練習問題

初級・中級・上級の自動採点問題。ヒント機能付き。

🗺 学習ロードマップ

順番に学習。前のレッスン完了で次を解放。

🏫 教師モード

学校向け。進捗確認とJSONエクスポート。
